# typetype Windows 使用说明书

## 当前使用版本

- 程序位置：以客户安装目录为准，安装后桌面会出现 `typetype` 快捷方式。
- 日志位置：软件设置页“关于/诊断”中会显示日志路径，默认仍保存在本机。
- 当前已配置：
  - 识别模式：流式输出
  - 翻译模型：HY-MT2-1.8B 本地模型，异常时自动回退 NLLB
  - 翻译目标：可在设置中选择英语、日语、德语、粤语等多语言
  - LLM 润写：选择大模型厂家后自动带出 Base URL 和推荐快速模型，客户只填 API Key
  - 开机自动启动：开启

## 快捷键

- `Ctrl + /` 或 `F8`：语音输入原文。
- `Ctrl + .` 或 `F9`：语音翻译。
- ALT 方案：右 Alt 语音输入，右 Alt + Shift 翻译。

说明：Windows 上不同输入法、键盘布局、远程桌面环境可能拦截 `Ctrl + /`、`Ctrl + .` 这类符号快捷键。本版本会同时注册备用键并增加热键健康检查；客户按符号键没反应时，直接用 `F8` 语音输入、`F9` 翻译，或在设置里点“ALT 方案”。

按一次开始录音，再按一次结束录音。

## 1. 流式原文输入测试

1. 打开记事本、浏览器输入框、微信输入框等可输入文字的位置。
2. 点击输入框，确保光标在输入框内。
3. 按 `Ctrl + /` 或 `F8` 开始说话。
4. 正常情况下，流式模式会边说边输出原文。
5. 再按 `Ctrl + /` 或 `F8` 停止录音。

本次修复后，Windows 流式音频 chunk 已统一降采样到 16kHz，避免“有录音 chunk 但识别不出原文”的问题。

流式原文输入会根据停顿自动断句：继续说下一段时自动补中文逗号，停止录音时自动补句号或问号。

## 2. 多语言翻译测试

1. 点击目标输入框。
2. 按 `Ctrl + .` 或 `F9` 开始录音。
3. 说一段普通话。
4. 再按 `Ctrl + .` 或 `F9` 停止录音。
5. 程序会先用非流式 SenseVoice 识别整段语音，再用 HY-MT2-1.8B 本地模型翻译成设置里选择的目标语言并粘贴。

说明：当前翻译不是逐字流式翻译。翻译模型需要拿到完整文本后再翻译，所以翻译请使用 `Ctrl + .`。在流式输出模式下，`Ctrl + .` 仍然可用，会自动切到翻译链路。

### 支持的翻译目标

设置里“翻译语言”可选：中文（简体）、英语、法语、葡萄牙语、西班牙语、日语、土耳其语、俄语、阿拉伯语、韩语、泰语、意大利语、德语、越南语、马来语、印尼语、菲律宾语、印地语、繁体中文、波兰语、捷克语、荷兰语、高棉语、缅甸语、波斯语、古吉拉特语、乌尔都语、泰卢固语、马拉地语、希伯来语、孟加拉语、泰米尔语、乌克兰语、藏语、哈萨克语、蒙古语、维吾尔语、粤语。

本地 HY-MT2-1.8B 模型和 llama.cpp 运行时已随安装包打包，断网也能翻译。若 HY-MT2 加载失败，程序会自动回退到原来的 NLLB 翻译模型。

## 3. LLM 结构化润写测试

LLM 润写适合在非流式语音输入中测试，因为流式输入优先保证原文实时输出。

1. 打开设置，把识别模式切换为“非流式”。
2. 确认“启用 LLM 润写”已开启。
3. 在“大模型厂家”下拉菜单中选择客户购买 API Key 的平台。
4. 软件会自动填好 Base URL 和推荐快速模型。
5. 客户只需要填入该平台自己的 API Key。
6. 点击“测试连接”，成功后设置会自动保存。

### 小白填写步骤

1. 先在“大模型厂家”选择平台，例如 `Kimi / 月之暗面国内版`。
2. “Base URL”和“模型”会自动填好，不要手动加 `/chat/completions`。
3. “API Key”只填客户在该厂商控制台创建的密钥。
4. 如果客户账号只开通了其他模型，可以把“模型”改成厂商控制台显示的模型 ID。
5. 点击“测试连接”。
6. 如果失败，优先检查三件事：Base URL 是否复制完整、API Key 是否属于同一个厂商、模型名是否在账号中可用。

### 结构化润写效果

1. 对输入框按 `Ctrl + /` 或 `F8`，说一段较长内容，例如：

   今天开会主要有三件事第一产品这周要把流式输入修好第二翻译功能粤语要继续测试第三我下周一之前整理使用说明另外风险是开机自启动可能在绿色版里注册表名称不稳定需要修复

2. 再按 `Ctrl + /` 或 `F8` 停止。

预期效果：短句会自动补标点；长段会自动梳理跳跃表达、修正明显口误或笔误，在不遗漏关键信息、数据和结论的前提下，整理为段落、清单、事项、风险、待办或“一、→（一）→1.”这类层级文档。

## 4. 开机自动启动测试

设置里开启“开机自动启动”后，Windows 注册表会写入：

`HKCU\Software\Microsoft\Windows\CurrentVersion\Run\typetype`

值应指向：

`C:\Users\51696\Desktop\typetype\typetype.exe --launch-at-login`

本次修复后，旧的 `electron.app.Electron` 和临时目录启动项会被清理。

## 5. 排查方法

- 如果没有粘贴文字，先确认目标输入框有光标。
- 如果快捷键没反应，打开设置查看快捷键是否注册成功。
- 如果翻译首次较慢，是 HY-MT2 本地模型正在加载；1.8B 模型约 1.13GB，普通 CPU 机器首次会有等待。
- 如果 LLM 润写失败，先看红字：401 多半是 API Key 或平台地区不匹配；400 多半是模型名或参数不支持；fetch failed 多半是网络、代理、防火墙或 TLS 证书问题。
- 详细错误请查看 `C:\Users\51696\Desktop\typetype\typetype.log`。

## 6. 本地词典说明

- 个人词典：客户自己添加、导入、启用或停用，保存在本机用户目录，不上传。
- 系统基础词库：内置只读词库，约 69 万条，来自人工整理安全词、jieba、THUOCL、OpenCC。
- 系统词库只用于术语保护、润写保留和翻译前提示，不会强制替换正常句子。
- 设置里可总开关系统基础词库，也可按分类启用或停用。
