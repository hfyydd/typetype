词典是做什么的？ 常用词用于保护专有名词，纠错词用于把“听错的词”自动改成正确写法，自动学习会在本机悄悄积累客户常说的专业词。
小白怎么用？ 先用“添加常用词”保存人名、品牌、项目名；识别总听错时再用“添加纠错词”。批量名单直接粘贴或导入表格。
添加常用词 常用词会被保护，润写和翻译时尽量不改错。
系统基础词库 系统基础词库为只读词库，仅用于术语保护和润写保留，不会上传，不会强制替换原文。

LLM 润写

调用 LLM API 优化语音转写结果，自动补全标点、梳理逻辑并整理层级结构。

傻瓜式填写：先选模型平台，软件会自动填 Base URL 和模型名；客户只粘贴该平台自己的 API Key。
国内版和国际版不能混用：例如 Kimi 国内平台 Key 要选择 Kimi / 月之暗面国内版，国际平台 Key 才选择 Kimi 国际版。
MiniMax、通义千问、百度千帆也区分地域地址，请按客户创建 API Key 的平台/地域选择。
小牛翻译 API 属于翻译接口，不适合放在结构化润写里；后续会放到“翻译 API”模块。
如果测试连接返回 401，通常是 API Key 填错、复制不完整、Key 所属平台不匹配，或账户额度/权限不可用；fetch failed 通常是网络、代理或 TLS 连接问题。
当前润写通道 未启用。

麦克风权限 允许应用采集麦克风音频。
辅助功能 用于回到原应用并执行自动粘贴。
输入监听 用于全局快捷键触发录音。

帮助中心

当前版本重点是"按一下开始录音，再按一下开始转写，成功后自动回填"。

1. Windows 语音输入可用 Ctrl+/ 或 F8，翻译可用 Ctrl+. 或 F9。

2. 按一下开始录音，再按一下开始转写。

3. 识别完成后会先写入剪贴板，再尝试自动回填到之前的输入框。

4. 如果快捷键没有反应，优先检查对应系统权限；如果不会自动回填，优先检查辅助功能权限。

导入方法说明

普通用户最简单的方式：一行一个常用词，直接粘贴或保存成 txt 后导入。

纠错词可以写成两列：第一列是可能识别错的词，第二列是正确词。Excel、WPS 表格、CSV、Word 表格都按这个规则识别。

WPS 的 .wps 和 .et 第一版请先另存为 .docx 或 .xlsx 再导入。
